Creating a Table

Tables allow developers to arrange text, images, and other data into rows and columns.
A basic table is created using the table tag, followed by a tr tag to create a new row in which a td tag is used to add data cells.

<table>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
        <td>Row 1, Cell 3</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
        <td>Row 2, Cell 3</td>
    </tr>
</table>

The above code will generate the following table.

Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

Note: A simple border and padding style has been applied to show table elements.

Adding Structure

Tables can be further structured with the addition of thead, tbody, and tfoot tags.
Headings for columns are identified with the thead tag. When naming columns, the td tag will be replaced with th.
The main data uses the tbody tag, and the summary of the table is displayed with the tfoot tag.
The following code shows a basic example of a table using all of the previously mentioned tags.

<table>
    <thead>
        <tr>
            <th>Column 1</th>
            <th>Column 2</th>`
            <th>Column 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
            <td>Row 1, Cell 3</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
            <td>Row 2, Cell 3</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
            <td>Footer 3</td>
        </tr>
    </tfoot>
</table>

Combine rows/columns

Using the colspan and rowspawn tags, it is possible to combine 2 or more cells.

<table>
    <thead>
        <tr>
            <th>Column 1</th>
            <th colspan="2">Column 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td rowspan="2">Row 1, Cell 2</td>
            <td>Row 1, Cell 3</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 3</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
            <td>Footer 3</td>
        </tr>
    </tfoot>
</table>
Column 1 Column 2
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 3
Footer 1 Footer 2 Footer 3

Captions

A caption or title can also be added to a table using the caption tag as shown below.

<table>
    <caption>Table Caption</caption>
    <thead>

        </tr>
